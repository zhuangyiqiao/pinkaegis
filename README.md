# 🌿 PINKAEGIS 个人管理系统

> Claude 打造的个人管理神器 · Notion 风格 · 专为 ADHD 友好设计

一个功能完整的个人管理网站，包含笔记、项目、日记、工作日志、能量追踪和个人档案等 6 大模块。所有数据保存在本地浏览器，无需账号，即开即用。

---

## ✨ 核心特性

### 🎯 6 个集成模块

| 模块 | 功能 | 快捷键 |
|------|------|--------|
| 📌 **学习笔记** | 卡片式笔记 + 标签分类 + 全文搜索 | 左侧菜单 |
| 📁 **项目记录** | 项目进度追踪 + 技能标签 + 状态管理 | 左侧菜单 |
| 📓 **每日杂记** | Brain Dump 快速记录 + 心情标签 + 时间轴 | 左侧菜单 |
| 💼 **实习记录** | 工作日志 + 学习总结 + 时间统计 | 左侧菜单 |
| ⚡ **能量追踪** | 睡眠、精力、专注度 + 7日趋势 | 左侧菜单 |
| 🌿 **关于我** | 个人档案 + 技能树 + 成就展示 | 左侧菜单 |

### 💡 设计理念

- **ADHD 友好** - 简洁界面，快速操作，无过度选项
- **Notion 风格** - 卡片设计，标签系统，灵活分类
- **100% 离线** - 所有数据保存在本地，零云服务依赖
- **即插即用** - 无需登录，打开即用，支持数据导出

## 📖 各模块使用指南

### 📌 学习笔记

**用于**：收集灵感、记录学习、建立知识库

```
1️⃣ 点击「+ 新建笔记」
2️⃣ 输入标题 + 内容
3️⃣ 选择或自定义标签
4️⃣ 点击「保存」
→ 笔记立即显示在卡片网格中，可即时搜索和过滤
```

**功能**：
- ✏️ 创建 / 删除笔记
- 🏷️ 多标签分类（预设：知识、想法、摘录、灵感）
- 🔍 实时搜索 + 标签过滤
- 💾 自动保存到浏览器本地

---

### 📁 项目记录

**用于**：追踪个人项目、展示作品、管理进度

```
1️⃣ 点击「+ 新建项目」
2️⃣ 填写项目信息（名称、描述、日期、进度）
3️⃣ 添加技能标签
4️⃣ 选择状态（计划/进行中/已完成）
5️⃣ 保存项目
→ 项目卡片显示，进度条实时更新
```

**功能**：
- 📊 0-100% 进度条（拖拽更新）
- 🏷️ 技能标签（多个，逗号分隔）
- 📅 日期范围管理
- 🔄 按状态过滤

---

### 📓 每日杂记

**用于**：思维放松、快速记录、情绪追踪

```

├─ TextArea 自由输入
```

**特色**：

### 💼 实习记录

**用于**：工作日志、学习总结、能力评估

## 💾 数据管理

### 存储位置

所有数据保存在浏览器 **LocalStorage**，无需账号、无云服务：

| 模块 | LocalStorage 键 | 容量 |
|------|-----------------|------|
| 笔记 | `pinkaegis_notes` | ~5KB/100条 |
| 项目 | `pinkaegis_projects` | ~3KB/10项目 |
| 日记 | `pinkaegis_journal` | ~10KB/200条 |
| 工作 | `pinkaegis_worklogs` | ~8KB/50条 |
| 能量 | `pinkaegis_energy` | ~2KB/30条 |
| 档案 | `pinkaegis_profile` | ~1KB |

### 数据安全

✅ **优点**：
- 完全离线，零服务器存储
- 即开即用，无隐私顾虑
- 多个浏览器可独立使用

⚠️ **注意**：
- 清除浏览器缓存会丢失数据
- 不同设备无法自动同步

### 数据备份

**建议**：定期导出个人档案（TXT 格式）

```
点击「🌿 关于我」→ 「📄 导出个人档案」
→ 生成 TXT 文件，保存到本地
```

## 🛠️ 技术栈

- **后端**：Flask + SQLite（用于留言系统）
- **前端**：原生 JavaScript（ES6+）+ CSS3
- **存储**：浏览器 LocalStorage
- **样式**：自定义 CSS（Notion 风格）

**不依赖任何 npm 包**（纯原生实现）

## 🐛 已知问题 & 限制

| 问题 | 说明 | 解决方案 |
|------|------|---------|
| 数据不同设备同步 | LocalStorage 无法跨设备 | 导出备份到 JSON/TXT |
| 大量数据时性能 | 1000+ 笔记可能显示缓慢 | 定期归档或升级后端 |
| 更新网页丢失输入 | 表单未提交时刷新页面 | 浏览器有自动恢复机制 |
